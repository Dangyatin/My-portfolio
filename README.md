# My-portfolio

Basic portfolio for Suleman Isma'il Isyaku. This repository contains a simple static portfolio site (index.html + styles.css).

To add your FreeCodeCamp certificate image so it displays on the site:

1. Create an `assets/` directory in the repository root.
2. Add the certificate image and name it `certificate.png` (or update `index.html` to match the filename).
